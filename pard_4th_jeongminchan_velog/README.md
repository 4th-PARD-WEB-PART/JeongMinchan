# Feed Page Project

## 개요

이 프로젝트는 사용자 피드를 표시하는 웹 애플리케이션입니다. 사용자는 다양한 포스트를 보고, 각각의 포스트에 좋아요를 클릭할 수 있습니다. 이 애플리케이션은 React와 Styled-components를 사용하여 개발되었습니다.

## 기술 스택

- React
- Styled-components
- JavaScript
